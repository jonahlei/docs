## 前提条件

获取你的 [API Key](https://console.seed.com/settings/keys)

## 调用 API

<Tabs>
  <Tab title="cURL">
    <Steps>
      <Step title="设置您的 API Key">
        从 [Seed 控制台](https://console.seed.com/settings/keys) 获取您的 API并将其设置为环境变量。

        ```bash
        export SEED_API_KEY='your-api-key-here'
        ```
      </Step>

      <Step title="进行您的第一次 API 调用">
        运行此命令来创建一个简单的照片解析工具：

        ```bash
          curl https://ark.cn-beijing.volces.com/api/v3/chat/completions \
            -H "Content-Type: application/json" \
            -H "Authorization: Bearer $ARK_API_KEY" \
            -d '{
              "model": "doubao-seed-1-6-250615",
              "messages": [
                  {
                      "content": [
                          {
                              "image_url": {
                                  "url": "https://ark-project.tos-cn-beijing.ivolces.com/images/view.jpeg"
                              },
                              "type": "image_url"
                          },
                          {
                              "text": "图片主要讲了什么?",
                              "type": "text"
                          }
                      ],
                      "role": "user"
                  }
              ]
          }'
        ````

  <Tab title="Python">
    <Steps>
      <Step title="设置您的 API Key">
        从 [Seed 控制台](https://console.seed.com/settings/keys) 获取您的 API Key 并将其设置为环境变量：

        ```bash
        export ARK_API_KEY='your-api-key-here'
        ```
      </Step>

      <Step title="安装 SDK">
        安装 Python SDK：

        ```bash
        pip install openai # 新安装 OpenAI SDK
        pip install --upgrade "openai" # 升级 OpenAI SDK
        ```
      </Step>

      <Step title="创建您的代码">
        将此保存为 `quickstart.py`：

        ```python
        import os
        from openai import OpenAI

        # 初始化Ark客户端，从环境变量中读取您的API Key，请确保您已将 API Key 存储在环境变量 ARK_API_KEY 中
        client = OpenAI(
            base_url="https://ark.cn-beijing.volces.com/api/v3",
            api_key=os.environ.get("ARK_API_KEY"),
        )

        response = client.chat.completions.create(
            # 指定模型 ID
            model="doubao-seed-1-6-250615",
            messages=[
                {
                    "role": "user",
                    "content": [
                        {
                            "type": "image_url",
                            "image_url": {
                                "url": "https://ark-project.tos-cn-beijing.ivolces.com/images/view.jpeg"
                            },
                        },
                        {"type": "text", "text": "图片主要讲了什么?"},
                    ],
                }
            ],
        )

        print(response.choices[0])
        ```
      </Step>

      <Step title="运行您的代码">
        ```bash
        python quickstart.py
        ```

        ```javascript
        import OpenAI from "openai";

        const openai = new OpenAI({
          apiKey: process.env.ARK_API_KEY,
          baseURL: "https://ark.cn-beijing.volces.com/api/v3",
        });

        async function main() {
          const response = await openai.chat.completions.create({
            model: "doubao-seed-1-6-250615",
            messages: [
              {
                role: "user",
                content: [
                  {
                    type: "image_url",
                    image_url: {
                      url: "https://ark-project.tos-cn-beijing.ivolces.com/images/view.jpeg",
                    },
                  },
                  { type: "text", text: "图片主要讲了什么?" },
                ],
              },
            ],
          });

          console.log(response.choices[0]);
        }

        main();
        ```
      </Step>

        ```bash
        node quickstart.js
        ```
